### About

**Gitset** is a suite of AI-assisted tools for everything around your code on GitHub Бқ” docs, issues, pull requests, releases. Every tool drafts a first pass; you refine it in plain language until it ships.

No subscription tiers. No token metering. No AI vendor lock-in. Bring your own key. Use it on the web or in your terminal. Inspect every line.

### The Suite

**Web** Бқ” [`gitset-dev/gitset`](https://github.com/gitset-dev/gitset)

- **README Generator** Бқ” full documentation drafts, refined until they read right
- **Issues Crafter** Бқ” structured, context-aware issue descriptions
- **PR Maker** Бқ” branch comparison, AI-written bodies, reviewers, labels, merge
- **Tags & Releases Manager** Бқ” releases with AI-drafted notes tied to real changes
- **Gitignore Builder** Бқ” stack-aware `.gitignore`
- **Repo Profiler** Бқ” AI-generated `description`, `website`, and `topics`
- **Backup Automator** Бқ” scheduled repository backups

**Templating** Бқ” Issues Crafter, PR Maker, README Generator, Tags & Releases Manager, and Gitignore Builder support custom templates. Define your format once, reuse everywhere.

<img src="https://raw.githubusercontent.com/gitset-dev/gitset/main/public/cli/favicon-48.png" height="20" align="center" /> **CLI** Бқ” standalone command-line companion

- `gitset commit` Бқ” AI commit messages *(CLI only)*
- `gitset gitignore` б· `gitset release` б· `gitset repo` б· `gitset labelspack` б· `gitset dependabot-resolver`
